# EC0249 Competency Mapping for SPA Content Structure

## Element 1: Identificar la situación/problema planteado

### Required Products (8 documents)

#### 1. Documento que describe el problema planteado
**Evaluation Criteria:**
- Incluye la afectación de la situación actual
- Establece el alcance
- Incluye la integración de la información obtenida
- Contiene la interpretación del problema y sus afectaciones

**SPA Components:**
- Problem description template with guided sections
- Impact analysis wizard
- Scope definition tool
- Information integration dashboard

#### 2. Afectación detectada de la situación actual
**Evaluation Criteria:**
- Es congruente con la integración de la información

**SPA Components:**
- Impact assessment matrix
- Current vs. desired state comparison tool
- Congruency validation checker

#### 3. Integración de la información presentada
**Evaluation Criteria:**
- Incluye la información recopilada
- Es congruente con el problema planteado por el consultante
- Incluye la interpretación de la información recopilada

**SPA Components:**
- Information repository with categorization
- Source tracking system
- Interpretation framework tool

#### 4. Reporte de metodología empleada
**Evaluation Criteria (8 components):**
- Incluye la definición de la situación y/o problema
- Incluye el establecimiento de un programa de entrevistas
- Incluye la identificación de las áreas involucradas
- Incluye el establecimiento de los estudios/pruebas a realizar
- Incluye el establecimiento de los requerimientos de información
- Incluye el establecimiento de un programa de observaciones de campo
- Incluye la búsqueda de información documental
- Contiene la forma en que evalúa la información obtenida

**SPA Components:**
- Methodology report builder with 8-section template
- Interview planning tool
- Area mapping interface
- Study design wizard
- Information requirements matrix
- Field observation planner
- Document search organizer
- Information evaluation framework

#### 5. Guía de entrevista empleada
**Evaluation Criteria:**
- Contiene el propósito de la entrevista
- Establece la solicitud de la descripción de actividades y responsabilidades
- Describe la información/documentación que se va a solicitar
- Incluye el cierre de la entrevista

**SPA Components:**
- Interview guide generator
- Purpose statement builder
- Activity/responsibility questionnaire designer
- Information request organizer
- Interview closure templates

#### 6. Cuestionario elaborado
**Evaluation Criteria (8 components):**
- Incluye la explicación del propósito del cuestionario
- Incluye espacio para los datos generales
- Menciona la confidencialidad de la información
- Contiene las instrucciones sobre la forma de llenado
- Establece preguntas relacionadas con la información buscada
- Especifica la documentación de soporte
- Contiene un espacio para comentarios finales
- Contiene frases de agradecimiento

**SPA Components:**
- Questionnaire design wizard with 8 mandatory sections
- Question type selector (open/closed)
- Confidentiality clause generator
- Instruction builder
- Documentation request organizer
- Comment section templates

#### 7. Programa de búsqueda de información documental
**Evaluation Criteria:**
- Incluye información interna relacionada con el problema
- Incluye información externa relacionada con el problema

**SPA Components:**
- Information search planner
- Internal/external source categorizer
- Search strategy builder
- Source reliability assessment tool

#### 8. Reporte de visita de campo
**Evaluation Criteria:**
- Muestra el objetivo de la visita de campo
- Establece el alcance
- Incluye la descripción de las observaciones realizadas
- Incluye el resultado de la visita de campo

**SPA Components:**
- Field visit report template
- Objective statement builder
- Scope definition tool
- Observation recording interface
- Results summarizer

### Performance Requirements
**Interview Execution (6 criteria):**
- Dar su nombre al inicio de la entrevista
- Indicar que la razón de la entrevista es obtener datos relativos al problema
- Solicitar que la información requerida se proporcione verbalmente/por escrito
- Solicitar que las evidencias que soporten sus respuestas sean proporcionadas
- Registrar las respuestas obtenidas
- Cerrar la entrevista agradeciendo la participación

**SPA Simulation Component:**
- Interview simulator with 6-step checklist
- Virtual client scenarios
- Response recording system
- Performance evaluation rubric

### Knowledge Requirements
1. **Entrevistas** (Comprensión)
   - Tipos de entrevistas

2. **Cuestionarios** (Comprensión)
   - Tipos de cuestionarios

3. **Indicadores** (Comprensión)
   - Concepto
   - Usos

4. **Fuentes de información** (Comprensión)
   - Tipos de fuentes de información
   - Características de las muestras

5. **Metodología de investigación de problemas** (Comprensión)
   - Definición de problemas
   - Requerimientos y evaluación de la información

**SPA Learning Components:**
- Interactive knowledge modules for each topic
- Self-assessment quizzes
- Practical application exercises
- Reference materials library

### Attitudes/Habits/Values
1. **Limpieza**: Información sin tachaduras y enmendaduras
2. **Orden**: Información apegada a la secuencia metodológica

**SPA Quality Control:**
- Document quality checker
- Methodology sequence validator
- Presentation standards enforcer

## Element 2: Desarrollar opciones de solución

### Required Products (2 documents)

#### 1. Reporte de las afectaciones encontradas
**Evaluation Criteria:**
- Describe la metodología aplicada
- Define las afectaciones encontradas
- Incluye la definición detallada de la situación a resolver

**SPA Components:**
- Impact report generator
- Methodology description template
- Situation definition tool

#### 2. Solución diseñada
**Evaluation Criteria:**
- Es congruente con la situación a resolver
- Menciona los beneficios de la solución
- Menciona las desventajas de la solución
- Cuenta con una justificación detallada
- Incluye las implicaciones de costo/beneficio

**SPA Components:**
- Solution design canvas
- Benefits/disadvantages analysis tool
- Justification builder
- Cost-benefit calculator

### Attitudes/Habits/Values
1. **Responsabilidad**: Acepta y firma lo estipulado en códigos de ética, confidencialidad y aviso de privacidad

**SPA Components:**
- Ethics agreement system
- Digital signature integration
- Privacy notice management

## Element 3: Presentar la propuesta de solución

### Required Products (5 documents)

#### 1. Propuesta de trabajo elaborada
**Evaluation Criteria (10 components):**
- Incluye los antecedentes y/o el diagnóstico
- Incluye la síntesis descriptiva del proyecto propuesto
- Especifica el alcance del proyecto propuesto
- Describe la solución propuesta en detalle
- Incluye un plan de trabajo
- Especifica los entregables por parte del consultor
- Especifica los riesgos del proyecto
- Especifica las responsabilidades del consultor
- Especifica las responsabilidades del consultante
- Especifica el costo estimado

**SPA Components:**
- Comprehensive proposal builder with 10 mandatory sections
- Project description wizard
- Scope definition tool
- Work plan generator
- Deliverable tracker
- Risk assessment matrix
- Responsibility assignment tool
- Cost estimation calculator

#### 2. Descripción detallada de la solución propuesta
**Evaluation Criteria:**
- Especifica las etapas de la instalación
- Describe el resultado esperado de cada etapa
- Especifica los indicadores de avance de cada etapa
- Menciona los mecanismos de control
- Describe los recursos a utilizar por parte del consultor
- Describe los recursos a utilizar por parte del consultante

**SPA Components:**
- Implementation roadmap builder
- Stage definition tool
- Progress indicator designer
- Control mechanism planner
- Resource allocation tool

#### 3. Plan de trabajo presentado en la propuesta
**Evaluation Criteria:**
- Menciona el resultado esperado de la solución a instalar
- Muestra las actividades a desarrollar
- Especifica los recursos a utilizar

**SPA Components:**
- Work plan template
- Activity scheduler
- Resource planning tool
- Expected outcomes tracker

#### 4. Actividades a desarrollar mencionadas en el plan
**Evaluation Criteria:**
- Presentan la calendarización de las actividades
- Mencionan el responsable de la actividad
- Presentan los indicadores de avance
- Especifican su mecanismo de control
- Especifican su mecanismo de seguimiento

**SPA Components:**
- Activity calendar/Gantt chart
- Responsibility assignment matrix
- Progress tracking system
- Control and monitoring dashboard

#### 5. Registro elaborado de los acuerdos alcanzados
**Evaluation Criteria (9 components):**
- Incluye la propuesta autorizada
- Menciona el alcance
- Menciona el resultado esperado
- Especifica el tiempo esperado de implantación
- Define las responsabilidades de los participantes
- Menciona el costo de la solución presentada
- Especifica las condiciones de pago acordadas
- Incluye una cláusula sobre confidencialidad
- Incluye una cláusula sobre la propiedad intelectual

**SPA Components:**
- Agreement record template with 9 mandatory sections
- Scope documentation tool
- Timeline tracker
- Payment terms builder
- Legal clause generator (confidentiality, IP)

### Performance Requirements
**Presentation Execution (11 criteria):**
- Describir la propuesta sugerida
- Mencionar el alcance
- Exponer las ventajas y desventajas
- Mencionar los responsables de parte del consultante
- Mencionar los responsables de parte del consultor
- Mencionar las etapas de la instalación
- Mencionar los entregables de cada etapa
- Mencionar las implicaciones de la implantación
- Describir los recursos a emplear
- Responder las preguntas o dudas expresadas
- Explicar las implicaciones del costo/beneficio

**SPA Simulation Component:**
- Presentation simulator with 11-point checklist
- Virtual client meeting environment
- Q&A simulation system
- Performance evaluation with feedback

### Attitudes/Habits/Values
1. **Limpieza**: Información sin tachaduras y enmendaduras
2. **Orden**: Información apegada a la secuencia metodológica

**SPA Quality Control:**
- Document quality verification
- Methodology sequence validation
- Professional presentation standards

### Progress Tracking Metrics
